# Curated Knowledge Map

## 목적

이 노트는 Vault 안에서 사람이 읽을 curated knowledge가 어디에 있는지 안내하는 router이다. raw archive 전체를 직접 열기 전에 이 노트와 아래 폴더를 먼저 확인한다.

## 현재 curated layer 상태

- 2025-03~2025-05: decisions/concepts/questions/reflections/daily 승격 완료
- 2025-06~2026-04: 월별 high-signal index, reflection, selected daily, 추가 concept/decision/question 승격 완료
- raw archive는 source of truth로 보존하고, 본문은 필요한 경우에만 targeted drill-down한다.

## 운영 원칙

1. raw 전체를 LLM context에 넣지 않는다.
2. compact index와 curated note를 먼저 읽는다.
3. 이력서 claim은 raw/project evidence 확인 전까지 `confidence: medium` 이하로 둔다.
4. Vault 변경 후 민감정보 스캔, graph 재생성, Git commit을 완료한다.
