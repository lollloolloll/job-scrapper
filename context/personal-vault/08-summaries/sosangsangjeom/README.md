## 전체 서사

소상상점은 사용자가 약 8개월 가까이 정성과 시간을 들인 장기 프로젝트다. AI 로그상 사용자의 관심은 다음 흐름으로 확장됐다.

```text
오류/명령어 질문
  → 기능 구현과 타입/라우팅 이해
  → API/인증/Storybook/빌드 문제 해결
  → GitHub PR 협업과 운영/SEO
  → 서비스 기획, 고객 확보, 포트폴리오/면접 서사화
```

## 이력서 핵심 문장

> 약 8개월간 소상공인 협업 플랫폼을 팀으로 개발하며, Next.js 기반 프론트엔드 구현뿐 아니라 GitHub 협업, API 연동, Storybook 컴포넌트 문서화, 빌드/배포 이슈 대응, SEO/운영 설정, 초기 사용자 확보 전략까지 경험한 프로젝트.
