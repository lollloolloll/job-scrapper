# Storybook/Vercel을 출시 품질 증거로 본다

## 결정

Storybook 배포와 Vercel 호스팅 기록은 단순 배포 경험이 아니라, 컴포넌트 검증과 사용자 접근 가능한 결과물을 만들었다는 출시 품질 증거로 다룬다.

## 이유

`스토리북 배포 설정`, `Vercel 호스팅 방법`, `Storybook decorator`, `thumbnail component` 같은 기록이 UI 검증/문서화/배포와 연결된다.

## 주의

최종 이력서 문장으로 쓰기 전에는 실제 배포 URL, story 파일, 스크린샷 또는 repo evidence를 확인한다.
