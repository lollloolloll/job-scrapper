# 2025년 4월 회고: 협업·컴포넌트·배포 품질로 확장

2025년 4월은 React/Next.js, PR 병합, 컴포넌트 리팩토링, Docker, SEO, Spring Security 등 실무형 질문이 증가한 달이다. 특히 2025-04-24, 2025-04-27, 2025-04-28, 2025-04-30은 협업과 프론트엔드 구조 이해가 강하게 드러난다.

- React vs Next.js, 컴포넌트 추상화, FSD에서 API 역할 같은 구조 질문이 나타난다.

- Docker/SEO/빌드 관련 질문은 배포 품질 관점으로 연결된다.
