# 선택지: 프론트엔드 개발자

## 판단

현재 가장 안전한 1순위 지원 직무다.

## 강한 근거

- 소상상점에서 Next.js/React/TypeScript 기반 사용자 웹 구현 경험.
- Storybook/Chromatic 기반 UI 컴포넌트 문서화.
- OAuth callback, token store, Axios API client 경험.
- SEO, sitemap, robots, middleware, build/deploy 문제 대응.
- GitHub branch/PR/merge/rollback 흐름 경험.
- 웹에이전시/MVP 프로젝트에서도 Next.js 사용자 웹과 랜딩/상담 흐름 경험이 있다.

## 리스크

- 단순 UI 구현자로 보이면 강점이 축소된다.
- 브라우저/React/Next.js 기본기 질문 대비가 필요하다.
- 프로젝트별 담당 범위와 정확한 성과를 확인해야 한다.

## 안전한 소개 문장

> Next.js/React 기반 사용자 웹 개발을 중심으로, API 연동, 인증 흐름, Storybook 컴포넌트 문서화, SEO/배포 이슈까지 경험한 프론트엔드 개발자입니다.

## 준비할 면접 주제

- React rendering, state, hooks
- Next.js App Router, SSR/SSG/CSR
- 인증/OAuth callback/token 관리
- API client와 interceptor
- Storybook을 쓰는 이유
- SEO/sitemap/robots/middleware
- Git branch/PR/revert/rollback
