# 11-career-strategy

진로 선택지, 직무 적합도, 지원 전략, 포지셔닝을 정리하는 폴더입니다.

## 먼저 읽을 노트

2. [[options/frontend-developer]] — 1순위 지원 직무.

## 원칙

- 프로젝트 근거 없이 직무 적합도를 단정하지 않는다.
- “안전하게 말할 수 있는 주장”과 “추가 확인이 필요한 주장”을 분리한다.
- AI 활용은 핵심 직함보다 지원 강점으로 다룬다.
- 비공개 고객/계약 프로젝트는 공개 범위를 명확히 표시한다.
